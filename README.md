This Game Rating System is my fifth C++ project and one of the most challenging so far, built to manage a list of games with names, ratings, and IDs using a custom struct and vector‑based storage. The program allows users to add, view, search, sort, and save game data, writing everything to a structured .txt file for persistence. This project pushed me to work with more complex logic, multi‑file organisation, and careful input handling while learning how to structure real data models and maintain clean, readable code. It marks a big step forward in my C++ learning journey as I move into more advanced, real‑world style programs.
